# GrooveMixDJ 🎧

Mixer virtual profissional para DJs — dois decks com EQ de 3 bandas, detecção
automática de BPM, controle via MIDI, playlist com Auto-DJ e forma de onda
interativa para scrub/seek em tempo real.

## Download

A versão mais recente está sempre disponível na aba [**Releases**](../../releases)
deste repositório.

## Versões

- **Free** — carregar músicas, play/pause, volume e crossfader. Sessão de
  demonstração de 1 hora.
- **PRO** — libera EQ de 3 bandas, detecção de BPM, controle MIDI e
  Playlist/Auto-DJ, sem limite de sessão. Licença anual.

Para adquirir a versão PRO, acesse: **[groovemixdj.vercel.app](https://groovemixdj.vercel.app)**

## Requisitos do sistema

- Windows 10 ou superior (64-bit)
- Placa de som com suporte a ASIO recomendado para menor latência
